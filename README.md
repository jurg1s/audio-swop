# Audio Swop

A simple tool for swapping audio tracks in video files.

## Installation 

To install Audio Swop, download the .deb package and run the following command:

```bash
sudo dpkg -i audio-swop_<version>_amd64.deb
```

To remove the tool:

```bash
sudo dpkg -r audio-swop
```
